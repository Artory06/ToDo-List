\# Todo List



Простое приложение для управления списком задач.

На текущем этапе работает как CLI (консольное приложение),

в будущем будет переписано в REST API на FastAPI.



\## Стек



\- Python 3.14

\- Стандартная библиотека (без внешних зависимостей)



\## Как запустить локально



\### Требования



\- Установленный Python 3.10+

\- Git



\### Установка



1\. Клонировать репозиторий:



```bash

&#x20;  git clone https://github.com/Artory06/ToDo-List.git

&#x20;  cd "ToDo List"

```



2\. Создать виртуальное окружение:



```bash

&#x20;  python -m venv .venv

```



3\. Активировать его:



&#x20;  - Windows (Git Bash): `source .venv/Scripts/activate`

&#x20;  - Windows (PowerShell): `.venv\\Scripts\\Activate.ps1`

&#x20;  - macOS / Linux: `source .venv/bin/activate`



4\. Установить зависимости:



```bash

&#x20;  pip install -r requirements.txt

```



\### Запуск



```bash

python main.py

```



Откроется меню в терминале:

```

1\. Добавить задачу

2\. Показать список задач

3\. Отметить задачу выполненной

4\. Удалить задачу

5\. Выход

```

\## Будущий REST API



После переписывания на FastAPI, приложение будет предоставлять REST API

для управления задачами. Ниже описан планируемый API контракт.



\### Модель задачи



```json

{

&#x20; "id": 1,

&#x20; "title": "Купить хлеб",

&#x20; "description": "В Пятёрочке",

&#x20; "is\_completed": false,

&#x20; "created\_at": "2026-04-14T10:30:00"

}

```



| Поле | Тип | Описание |

|---|---|---|

| `id` | int | Уникальный идентификатор задачи |

| `title` | string | Название задачи (обязательное) |

| `description` | string \\| null | Описание (опциональное) |

| `is\_completed` | bool | Выполнена ли задача |

| `created\_at` | string (ISO 8601) | Время создания |



\### Список эндпоинтов



| Метод | URL | Описание |

|---|---|---|

| `GET` | `/tasks` | Получить все задачи (с возможностью фильтрации) |

| `GET` | `/tasks/{id}` | Получить задачу по id |

| `POST` | `/tasks` | Создать новую задачу |

| `PUT` | `/tasks/{id}` | Обновить задачу |

| `DELETE` | `/tasks/{id}` | Удалить задачу |



\---



\### `GET /tasks` — получить все задачи



\*\*Query-параметры (опционально):\*\*



| Параметр | Тип | Описание |

|---|---|---|

| `is\_completed` | bool | Фильтр по статусу выполнения |



\*\*Пример запроса:\*\*



```

GET /tasks?is\_completed=false

```



\*\*Ответ `200 OK`:\*\*



```json

\[

&#x20; {

&#x20;   "id": 1,

&#x20;   "title": "Купить хлеб",

&#x20;   "description": "В Пятёрочке",

&#x20;   "is\_completed": false,

&#x20;   "created\_at": "2026-04-14T10:30:00"

&#x20; },

&#x20; {

&#x20;   "id": 2,

&#x20;   "title": "Помыть посуду",

&#x20;   "description": null,

&#x20;   "is\_completed": false,

&#x20;   "created\_at": "2026-04-14T11:00:00"

&#x20; }

]

```



\---



\### `GET /tasks/{id}` — получить задачу по id



\*\*Path-параметры:\*\*



| Параметр | Тип | Описание |

|---|---|---|

| `id` | int | Идентификатор задачи |



\*\*Пример запроса:\*\*



```

GET /tasks/1

```



\*\*Ответ `200 OK`:\*\*



```json

{

&#x20; "id": 1,

&#x20; "title": "Купить хлеб",

&#x20; "description": "В Пятёрочке",

&#x20; "is\_completed": false,

&#x20; "created\_at": "2026-04-14T10:30:00"

}

```



\*\*Ответ `404 Not Found`\*\* (задачи нет):



```json

{

&#x20; "detail": "Задача 1 не найдена"

}

```



\---



\### `POST /tasks` — создать задачу



\*\*Тело запроса:\*\*



```json

{

&#x20; "title": "Купить хлеб",

&#x20; "description": "В Пятёрочке"

}

```



| Поле | Тип | Обязательное |

|---|---|---|

| `title` | string | да |

| `description` | string | нет |



\*\*Ответ `201 Created`:\*\*



```json

{

&#x20; "id": 5,

&#x20; "title": "Купить хлеб",

&#x20; "description": "В Пятёрочке",

&#x20; "is\_completed": false,

&#x20; "created\_at": "2026-04-14T10:30:00"

}

```



\*\*Ответ `422 Unprocessable Entity`\*\* (нет обязательного поля или неверный тип):



```json

{

&#x20; "detail": \[

&#x20;   {

&#x20;     "loc": \["body", "title"],

&#x20;     "msg": "field required",

&#x20;     "type": "value\_error.missing"

&#x20;   }

&#x20; ]

}

```



\---



\### `PUT /tasks/{id}` — обновить задачу



\*\*Path-параметры:\*\*



| Параметр | Тип | Описание |

|---|---|---|

| `id` | int | Идентификатор задачи |



\*\*Тело запроса:\*\*



```json

{

&#x20; "title": "Купить молоко",

&#x20; "description": "В Магните",

&#x20; "is\_completed": true

}

```



\*\*Ответ `200 OK`:\*\*



```json

{

&#x20; "id": 1,

&#x20; "title": "Купить молоко",

&#x20; "description": "В Магните",

&#x20; "is\_completed": true,

&#x20; "created\_at": "2026-04-14T10:30:00"

}

```



\*\*Ответ `404 Not Found`\*\* (задачи нет):



```json

{

&#x20; "detail": "Задача 1 не найдена"

}

```



\---



\### `DELETE /tasks/{id}` — удалить задачу



\*\*Path-параметры:\*\*



| Параметр | Тип | Описание |

|---|---|---|

| `id` | int | Идентификатор задачи |



\*\*Ответ `204 No Content`\*\* (тела ответа нет).



\*\*Ответ `404 Not Found`\*\* (задачи нет):



```json

{

&#x20; "detail": "Задача 1 не найдена"

}

```



\---



\### Сводная таблица кодов ответов



| Код | Когда |

|---|---|

| `200 OK` | Успешное чтение или обновление |

| `201 Created` | Успешное создание |

| `204 No Content` | Успешное удаление |

| `404 Not Found` | Задача с указанным id не найдена |

| `422 Unprocessable Entity` | Неверные данные в теле запроса |

| `500 Internal Server Error` | Внутренняя ошибка сервера |
